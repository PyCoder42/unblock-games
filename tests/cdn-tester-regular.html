CDNs that can serve config.json from a GitHub repo

  • jsDelivr (current) — serves files via cdn.jsdelivr.net/gh/user/repo@tag/file
  • Statically — serves via cdn.statically.io/gh/user/repo/tag/file
  • GitHack — serves via rawcdn.githack.com/user/repo/tag/file
  • GitHub raw — direct raw content via raw.githubusercontent.com (no CDN caching)

CDNs that require npm publishing

  • unpkg — serves npm packages only; this project can use a published config package there
  • esm.sh — ES module CDN for npm packages
  • Skypack — ESM CDN for npm packages
  • cdnjs — curated library CDN; requires submitting a library

How to switch providers

  • Edit .secure-config to update the CDN URL template
  • Run ./Formatting Scripts/sync-protocol-versions.sh --force to regenerate all secure files
  • If you switch to an npm-backed fallback, publish config-pkg/ before relying on it
